# Back-End-Inventory

Project Intern Back End at RPL GDC Laboratory

## Requirement
npm install  
npm install express  
npm install body-parser  
npm install jsonwebtoken  
npm install bcryptjs  
npm install @hapi/joi  
npm install --save-nodemon  
npm install dotenv
npm install mongoose
npm install nanoid
